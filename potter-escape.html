About this Page

This page features a game where Harry Potter must escape from spiders and Dementors. The game uses Motion UI to create a magical experience with animated elements and smooth transitions.

Magic

Animations enhance the challenge with moving obstacles and dynamic gameplay, bringing the Forbidden Forest to life.

Spell List

  • Harry's flying animation (jumping)
  • Obstacle appearance/disappearance
  • Seamless scrolling background
  • Collision detection with spiders and Dementors
  • Responsive design for all screen sizes

Potter Escape

Help Harry escape from the dangers of the Forbidden Forest!

Score : 0

Instructions:


  1. Press the START button to begin the game.
  2. Press ENTER to make Harry jump over spiders and avoid Dementors.
  3. Earn points by surviving longer!
